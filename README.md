# Sequential Bayes factor testing: a new framework for decision fusion

## Introduction

This code implements the results published in Parras, J., & Zazo, S. (2019, July). Sequential Bayes factor testing: a new framework for decision fusion. In 20th International workshop on Signal processing advances in wireless communications (SPAWC), 2019 , (pp. 1-5). IEEE. [DOI](https://doi.org/10.1109/SPAWC.2019.8815408).
